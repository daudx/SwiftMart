# SwiftMart — E-Commerce Mobile Application

[![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)](https://flutter.dev)
[![Firebase](https://img.shields.io/badge/firebase-%23039BE5.svg?style=for-the-badge&logo=firebase)](https://firebase.google.com)
[![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)](https://dart.dev)

SwiftMart is a premium, fast, and intuitive mobile e-commerce platform designed to provide a seamless shopping experience. Built with **Flutter** and **Firebase**, it focuses on speed, reliability, and a clean user interface.

---

## 🚀 Overview
The SwiftMart project aims to bridge the gap between users and products through a modern mobile interface. It integrates essential e-commerce features, from secure authentication to a simulated checkout process, ensuring a smooth journey for every shopper.

## ✨ Key Features

### 👤 User Features
- **User Authentication**: Secure sign-up, login, and password recovery.
- **Product Catalog**: Explore a wide range of products with detailed descriptions, high-quality images, and real-time pricing.
- **Shopping Cart**: Dynamic cart management with automatic total price calculations.
- **Simulated Checkout**: A realistic checkout flow including shipping detail input and a simulated payment gateway.
- **Order History**: Keep track of past purchases and their current statuses.
- **User Profile**: Personalized profile management and access to shopping history.
- **Push Notifications**: Stay updated with the latest sales, discounts, and order statuses.

### 🛠️ Administrative Features
- **Admin Panel**: Comprehensive dashboard for administrators to add, update, or remove products from the catalog.

## 🛠️ Technology Stack

- **Backend/Database**: [Firebase](https://firebase.google.com) (Firestore, Authentication, Cloud Messaging)

## ⚙️ Getting Started
To get a local copy up and running, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/daudx/SwiftMart.git
   ```
2. **Install dependencies**:
   ```bash
   flutter pub get
   ```
3. **Run the app**:
   ```bash
   flutter run
   ```
